1. Formato de un HTML

Todos los documentos HTML tienen el mismo formato basico que es el siguiente

   <!DOCTYPE html>
   <html lang="es">
        <head>
            <title> Titulo de la pagina</title>
            <meta charset="utf-8"/>
            <meta name ="author" content="Cristian">
        </head>
        <body> El body representa el contenido principal de un documento hmtl</body>
    </html>

Este enlace te lleva a una pagina basica de html


2. Elementos de un HTML

Los distintos elementos HTML son las partes que componen los documentos HTML y se pueden clasificar de la siguiente manera


2.1 Elemento Raíz

    
    
<!DOCTYPE html>
	
    Define que el documento está bajo el estándar de HTML 5 y ha de ser ñlo primero que ha de aparecer en el documento HTML.
    
    En HTML se veria de la siguiente manera:

    <!DOCTYPE html>


    
<html></html>
	
    Representa la raíz de un documento HTML o XHTML. Todos los demás elementos deben ser descendientes de este elemento.

2.2 Metadatos

    
<head></head>
    
    Elemento en el que se almacenan los metadatos acerca del documento, incluyendo enlaces a, o definiciones de, scripts y hojas de estilo.
     
    
<title></title>
  
    Define el título del documento, el cual se muestra en la barra de título del navegador o en las pestañas de página.

    
<meta/>
          
    Define los metadatos que no pueden ser definidos usando otro elemento HTML.

        <meta charset="utf-8"/>
    
    
<style></style>
  
    Etiqueta de estilo usada para escribir CSS en línea.

    
<link></link>

2.3 Secciones

    
<body></body>
         
    Representa el contenido principal de un documento HTML. Solo hay un puede haber uno por documento.

    
<section></section>
   
    Define una sección en un documento.

    
<nav></nav>
           
    Define una sección que solamente contiene enlaces de navegación.

    
<article></article>
   
    Define contenido autónomo que podría existir independientemente del resto del contenido.

    
<aside></aside>
       
    Define algunos contenidos vagamente relacionados con el resto del contenido de la página. Si se quita, el contenido restante seguirá teniendo sentido.

    
<h1></h1>
             
    Los elementos de tipo cabecera implementan seis niveles, h1 es el de mayor importancia y h6 el de menor importancia. Se usan sobre todo para titulos.    

    
<header></header>
     
    Elemento que se usa para definir la cabecera de una página o de una sección.    

    
<footer></footer>
     
    Elemento que se usa para definir pies de página o secciones.    

    
<address></address>
   
    Define una sección que contiene información de contacto.

2.4 Agrupaciones

    
<p></p>
           
    Elemento apropiado para distribuir el texto en párrafos.    

    
<hr>
              
    Elemento que representa un cambio de tema entre parrafos con una linea.

    
<pre></pre>
       
    Elemento que se usa para escribir texto preformateado.

    
<ol></ol>
         
    Elemento que permite crear listas ordenadas.

    
<ul></ul>
         
    Elemento que permite crear listas no ordenadas.  

    
<li></li>
         
    Elemento que declara cada uno de los puntos de una lista ordenada.

    
<dl></dl>
         
    Elemento que representa una lista descriptiva.  
    
    
<dt></dt>
         
    Elemento que especifica un término en una en una descripción y debe ser usado dentro de un elemento <dl>

    
<dd></dd>
         
    Elemento que provee detalles acerca de un término precedente <dt> en una lista <dl>

    
<div></div>
       
    Elemento que sirve para crear secciones o agrupar contenidos.

2.5 Semánticos


    
<a></a>
           
    Elemento que crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página.

    
<em></em>
        
    Elemento apropiado para marcar con énfasis las partes importantes de un texto. Puede anidarse y cada vez el énfasis es mayor.

    
<strong></strong>
 
    Elemento que al igual que el elemento <em> se usa para marcar con énfasis partes importantes de un texto.

    
<small></small>
   
    Elemento que hace el tamaño del texto una talla más pequeña que el tamaño mínimo de fuente del navegador.

    
<sub></sub>
       
    Elemento que define un fragmento de texto que se debe mostrar más bajo y generalmente más pequeño que el resto del texto.

    
<sup></sup>
       
    Elemento que define un fragmento de texto que se debe mostrar más alto y generalmente más pequeño que el resto del texto.
    
    
<i></i>
           
    Elemento que transforma un fragmento de un texto en cursiva o italica.

    
<b></b>
           
    Elemento que aplica negrita a un fragmento del texto.

    
<u></u>
           
    Elemento que aplica subrayado a un fragmento del texto.

    
<span></span>
     
    Elemento que sirve para aplicar estilo o para agrupar elementos en linea.

    
<br>
              
    Elemento que produce un salto de linea en el texto.

2.6 Encajados


    
<img>
                     
    Elemento que permite insertar una imagen en el documento.

    
<iframe></iframe>
         
    Elemento que permite incrustar otra página HTML en la página actual.

    
<embed src="" type="">
    
    Elemento que representa un punto de integración para una aplicaciñon externa.

    
<object></object>
         
    Elemento que representa un recurso externo.

    
<video></video>
           
    Elemento que permite incrustar videos en un documento HTML.

    
<audio></audio>
           
    Elemento que permite incrustar contenido de audio en un documento HTML.

    
<source>
                  
    Elemento que especifica recursos de medios múltiples para los elementos <picture>, <audio>, <video>.

    
<track>
                   
    Elemento que se utiliza como elemento secundario de los elementos multimedia <audio> y <video>.

<tbody></tbody>
       
    Elemento que encapsula el contenido del cuerpo de una tabla.
    
    
<thead></thead>
       
    Elemento que define un conjunto de filas para que sean el encabezado de las columnas de la tabla.

    
<tfoot></tfoot>
       
    Elemento que define un conjunto de filas para que sean el pie de página de las columnas de la tabla.

    
<caption></caption>
   
    Elemento que especifica el titulo de una tabla.

3. Documento CSS
CSS es un lenguaje de diseño gráfico para definir y crear la presentación de un documento 
    estructurado escrito en un lenguaje de marcado.

3.1 Sintaxis CSS
        
La estructura de una regla CSS sería la siguiente:

    
        Selector{ propiedad: valor de la propiedad;}

        
Descripción de las partes


        Selector
        Es el elemento HTML al que se aplicará el estilo.

        Propiedades
        Las distintas maneras de dar estilo a un elemento HTML.

        Valor de la propiedad
        Determina uno de los muchos valores que se pueden usar con una propiedad.
        

        Otras partes importantes:

        Todas las reglas deben tener encapsuladas la propiedad y el valor de la propiedad entre {}.
        Se deben usar los dos : para separar una propiedad de su valor.
        Dentro de cada regla, debes usar el punto y coma (;) para separar una declaración de la siguiente.


        También puedes usar varios selectores separados por comas para aplicarles una regla de la siguiente manera.

        Selector,selector,selector {propiedad: valor de la propiedad;}


        
3.2 Selectores CSS


        Hay varios tipos de selectores aparte del selector de Listas como hemos visto,
        ahora veremos otros:


        Selector de identificación
        El elemento ha de tener un id específico y único.
        por ejemplo:
            <p id="#Prueba">Texto de prueba</p>

        Selector de clase
        Al elemento se le ha de asignar una clase que puede valer para otros Listas:
        Un ejemplo:
            <p class="Prueba">Texto de prueba</p>

        Selector de atributo
        Se aplica el selector al elemento en línea.
        Un ejemplo:
            <img src="">

        Selector de Pseudo-clase
        Solo se usan en ciertos Listas especificos 
        Un ejemplo:
            a:hover

        

    
3.3 Fuentes y texto

    
    Las fuentes se pueden aplicar a todos los textos incluidos los enlaces. También tienen 
    propiedades específicas las cuales, se describen a continuación:

    Font-family
    Esta propiedad se usa para determinar la fuente que se quiere usar para representar el texto.
    Se puede indicar más de una por si no se tiene instalada la primera.
    
    Un ejemplo:
        p{font-family: Arial,Verdana,Helvetica,sans-serif;}


    Font-style
    Esta propiedad se usa para indicar el estilo de la fuente.

    Un ejemplo:
        p{font-style:italic;}
    

    Font-weight
    Esta propiedad se puede definir el grosor de la letra.
    El peso puede especificarse de las siguientes 3 maneras:

    Absolutas
    Con los valores Normal y bold

    Relativa
    Con los valores bolder y lighter

    Especifica
    Aquí los valores se especifican con números concretos
    
    Varios ejemplos:
        p{font-weight:bold;} 

        p{font-weight:bolder;} 
        
        p{font-weight:800;} 
    
    font-align
    Propiedad que se usa para alinear el texto.

    Un  ejemplo:    
            p{ text-align:justify;}

    Text-decoration
    Ha esta propiedad se le pueden aplicar los valores none, overline, underline, through y blink
    para decorar el texto de distintas maneras.

    Un  ejemplo:
        p{text-decoration:throught; color:red;}

    text-indent
    Con esta propiedad se puede aplicar una sangría al texto.
    
    Un ejemplo:
    p{ text-indent:50px;}


    
3.4 Cajas

    El modelo de cajas es el comportamiento de CSS que hace que todos los Listas de las páginas se representen mediante cajas rectangulares.

    Existen dos propiedades para definir el tamaño de una caja, height y width

    Height definirá la altura y width la anchura

    

    
    
    Aparte de estas existen Margin, Padding y Border que sirven para organizar mejor las cajas.
    
    Margin establece los márgenes para un elemento.
    
    Padding establece el espacio de relleno entre el contenido del elemento y el borde.

    Border permite especificar el estilo, el ancho, y el color del borde de un elemento.

3.5 Fondo de objeto

    
    El modelo de caja tambien permite cambiar el fondo de las cajas con distintas propiedades que 
    veremos a continuación:

    Background-color
    
    Esta propiedad permite mostrar un color de fondo sólido en la caja de un elemento.

    Background-image
    
    Propiedad que permite mostrar una imagen como fondo de la caja de cualquier elemento.    

    Background-repeat
    
    Esta propiedad permite controlar la forma de repetición de las imágenes de fondo.

    Background-position

    Propiedad que permite permite controlar la posición de la imagen dentro del fondo del elemento.

    Background-attachment
    
    Propiedad que hace que el fondo permanezca fijo cuando la ventana del navegador se desplaza mediante las barras de scroll.

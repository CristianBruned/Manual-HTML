2. Elementos de un HTML

Los distintos elementos HTML son las partes que componen los documentos HTML y se pueden clasificar de la siguiente manera


2.1 Elemento Raíz

<!DOCTYPE html>

Define que el documento está bajo el estándar de HTML 5 y ha de ser lo primero que ha de aparecer en el documento HTML.

<html></html>

Representa la raíz de un documento HTML o XHTML. Todos los demás elementos deben ser descendientes de este elemento.
2.2 Metadatos

Los metadatos son elementos HTML que contienen información sobre la propia página web.


<head></head>

Elemento en el que se almacenan los metadatos acerca del documento, incluyendo enlaces a, o definiciones de, scripts y hojas de estilo.

<title></title>

Define el título del documento, el cual se muestra en la barra de título del navegador o en las pestañas de página.

<meta/>

Define los metadatos que no pueden ser definidos usando otro elemento HTML.

<style></style>

Etiqueta de estilo usada para escribir CSS en línea.

<link></link>

Elemento usado para enlazar JavaScript y/o CSS externos con el documento HTML al que se le quiera aplicar.

Los atributos más relevantes son:

El atributo href el cual especifica la URL del recurso enlazado. La URL puede ser absoluta o relativa.
El atributo rel indica la relación del documento enlazado con el actual.
El atributo target específica una ventana o frame en el que se abrira el documento cargado.
El atributo type específica el tipo del archivo enlazado.

2.3 Secciones

<body></body>

Representa el contenido principal de un documento HTML. Solo puede haber uno por documento.

<section></section>

Define una sección en un documento.

<nav></nav>

Define una sección que solamente contiene enlaces de navegación.
Ejemplo:
W3S I.E.S

<article></article>

Define contenido autónomo que podría existir independientemente del resto del contenido.

<aside></aside>

Define algunos contenidos vagamente relacionados con el resto del contenido de la página. Si se quita, el contenido restante seguirá teniendo sentido.
Ejemplo:

Texto de ejemplo

Ejemplo

Frase, acción u objeto que se usa para explicar, ilustrar o aclarar una cosa.

<h1></h1>

Los elementos de tipo cabecera implementan seis niveles, h1 es el de mayor importancia y h6 el de menor importancia. Se usan sobre todo para titulos.

Ejemplo:

Titulo 1

Titulo 2

Titulo 3

Titulo 4

Titulo 5
Titulo 6

<header></header>

Elemento que se usa para definir la cabecera de una página o de una sección.

<footer></footer>

Elemento que se usa para definir pies de página o secciones.

<address></address>

Define una sección que contiene información de contacto.

Ejemplo:
e-mail:65350@gmail.com

2.5 Semánticos

<a></a>

Elemento que crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página.

El codigo sería el siguiente: <a href="http://iessierradeguara.com/wordpress/">I.E.S Sierra de Guara</a>
Ejemplo: I.E.S Sierra de Guara

Los atributos más relevantes son:

El atributo href el cual especifica la URL del recurso enlazado. La URL puede ser absoluta o relativa.
El atributo rel indica la relación del documento enlazado con el actual.
El atributo target específica una ventana o frame en el que se abrira el documento cargado.
El atributo type específica el tipo del archivo enlazado.

<em></em>

Elemento apropiado para marcar con énfasis las partes importantes de un texto. Puede anidarse y cada vez el énfasis es mayor.

El codigo sería el siguiente:

Ejemplo:

Ejemplo:

Ejemplo:

Ejemplo:

2.6 Encajados

Los elementos encajados son elementos HTML que permiten insertar en la página web distintos tipos de elementos .

<img>

Elemento que permite insertar una imagen en el documento.

La manera de incrustarlo es la siguiente:
<img src="Dirección de la imagen">

<iframe></iframe>

Elemento que permite incrustar otra página HTML en la página actual.

La manera de incrustarlo es la siguiente:
<iframe src="Dirección de la página"></iframe>

<object></object>

Elemento que representa un recurso externo.

La manera de incrustarlo es la siguiente:
<object data="Dirección del elemento" type="Tipo de la aplicación"></object>

<video></video>

Elemento que permite incrustar videos en un documento HTML.

La manera de incrustarlo es la siguiente:
<video width="Tamaño" controls> <source src="Ruta del video" type="Tipo del video"> </video>

<source>

Elemento que especifica recursos multimedia alternativos para los elementos <picture>, <audio>, <video>.

<track>

Elemento que se utiliza para especificar los subtitulos u otros archivos que contienen texto, los cuales deberian ser visibles cuando se ejecuten los elementos <audio> y <video>.

<embed>

Elemento que representa un punto de integración para una aplicacion externa que no es HTML.

Elemento que define un conjunto de filas para que sean el encabezado de las columnas de la tabla.

<tbody></tbody>

Elemento que encapsula el contenido del cuerpo de una tabla.

<tfoot></tfoot>

Elemento que define un conjunto de filas para que sean el pie de página de las columnas de la tabla.

El Elemento <th> define una celda como la cabecera de una columna de una tabla.

El Elemento <caption> es el que especifica el titulo de una tabla.
